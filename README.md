# 📋 Project Management Tool — ATM Mobilis

A full-stack internal web application for project and task management, 
developed during a 6-month internship at ATM Mobilis (Algeria's leading telecom operator).

> Built with React.js + ASP.NET Core Web API + SQL Server

---

## 🚀 Tech Stack

**Backend:**
- C# / ASP.NET Core 8 Web API
- Entity Framework Core
- SQL Server (ProjectManagementDB — 13 tables)
- JWT Authentication + BCrypt
- Swagger  for API testing

**Frontend:**
- React.js + React Router DOM
- Axios (with JWT interceptors)
- Mobilis brand identity (green & white)

**Architecture:** 3-tier (Presentation / Business / Data)

---

## ✨ Features

### 👤 Authentication
- Secure JWT login (24h token)
- Role-based access: Manager / Chef de Projet / Membre

### 📊 Dashboard
- Overview of active projects, tasks, completions & validations
- Personalized welcome per role

### 📁 Project Management
- Create, assign and track projects
- Real-time progress monitoring

### ✅ Task Management
- Task assignment to team members
- Progress tracking (0-100%)
- Double validation system (developer + project manager)
- Priority & deadline management

### 📈 Reporting
- Project and team performance reports
